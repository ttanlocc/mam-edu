# IELTS 8.5 — 52-Week Curriculum Map (SEQUENTIAL-FOCUS · mapped to DOWNLOADED materials)

Every cell cites an item from `material-inventory.md` (the real archive in `laged-app`). Spec Rule #1 —
closed corpus. Authoring agents expand each row into daily blocks.

**Plan shape (user decision 2026-07-18):** SEQUENTIAL, not parallel.
- **Months 1–6 (w1–26): NO WRITING AT ALL.** Grind **L & R → 9.0** (the two receptive skills get ~145′/day)
  and build **Speaking 0 → 7.0** (italki diagnostic w4, 1×/wk from w5, 2×/wk from w14).
- **Months 7–12 (w27–52): Writing 0 → 7.0** as the largest daily block (95–100′), **Speaking 7.0 → 8.0**,
  L/R drop to 30–35′/day maintenance.
- Band math: **L9 + R9 + W7.0 + S8.0 = 33 → overall 8.5** (zero-slack).
- Hard gates: **w13** (L+R+S phase mock) · **w26** (L/R 39–40/40 + S 7.0 — condition to open Writing) ·
  **w30** (first Writing marking) · **w34** (weekly marking on) · **w39** (W ~6.5 · S 7.5 · first full 4-skill mock) ·
  **w47** (W 7.0 confirmed · S 8.0).

**Cite short-forms:** `NB`=NgọcBách · `NB-R#`=NgọcBách Reading Online folder · `NB-W`=Writing Online ·
`NB-V#`=Vocabulary online Chủ đề # · `CCT-I/CCT-A`=ChépChínhTả Intermediate/Advanced (lesson N) ·
`BBC#`=NguyễnHuyền BBC ep · `NH-72 #`=NguyễnHuyền 72-track Track # · `NP#`=NghĩaPhan Test # ·
`RtI#`=RoadToIELTS part # · `FB`=Writing marking loop · `iTk`=Speaking tutor. **Sat (T7)** = mock/review.
**Sun** = rest. Gap-fillers (band-8 models, grammar, pronunciation course) are *optional*, flagged `[opt]`,
never required daily material.

### Invariants baked into this map
- **Every cell ∈ material-inventory.md** (real downloaded archive). No Cambridge-book / Simon / Hancock / Collins refs (not downloaded).
- **w1–26 contain ZERO Writing work** — no Writing blocks, no essay tasks hidden in other skills. The NB Writing course is untouched until w27.
- **w27–39 consume the finite NB Writing content in order**: Task 1 ×6 (w27–30), Task 2 ×4 (w31–34), BONUS ×4 (w35–37) — same lesson pattern (video → model → self-write → self-review) the old Phase 1 used, now compressed with double the daily minutes.
- **Feedback timing:** iTk diagnostic w4 · iTk 1×/wk w5–13 · iTk 2×/wk w14–52 · scored S mocks w13/w20/w26/w39/w47 · Writing marking first at w30, weekly from w34. From zero, W/S cannot self-grade — the loops are load-bearing, not optional.
- **L/R front-loaded**: the full-test banks (NB-R10 Cambridge/BC, NP1–26) carry the w1–26 grind; P3–4 reuse NP1–9 (fresh eyes ≥6 months later), RtI1–9 and the BC bank for maintenance + weekly full mocks. Reuse/review acceptable — the point is precision, not novelty.
- **Speaking is the only 12-month skill** — its 8.0 target leans on that continuity (receptive 9.0 base + tutor loop). Fluency work never pauses, even in test week.
- **`[opt]` gap-fillers** (band-8 Writing models, grammar top-up, pronunciation course) appear only as optional accelerators, never as required daily material — the plan is runnable 100% on the downloaded archive, with FB/iTk carrying W to 7.0 and S to 8.0.
